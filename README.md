# 🧬 ProteinViz

**Version:** 1.0.0

> A Python-based toolkit for exploring and analysing protein structures using Biopython.

---

## 📖 Overview

ProteinViz is a computational biology project that demonstrates how to analyse protein structures obtained from the Protein Data Bank (PDB).

The project uses Biopython to extract structural information such as protein chains, residues, and atoms, and presents the results through summary reports and visualizations.

---

## 🎯 Objectives

- Parse protein structure files (PDB format)
- Count chains, residues, and atoms
- Analyse amino acid composition
- Generate summary reports
- Create scientific visualizations
- Learn structural bioinformatics using Python
